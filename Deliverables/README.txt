------Instalation-------
In order to use this app you need Visual studio 2015/2017. 
You can import the project and run it.
You also need MySql server and MySql Workbench where the 
mysql script provided will be used to create the data base.

------Users-------------
This app can be used by 2 users(admin and ordinary user).
The system detects what users connects(a different window will appear for
different users).
